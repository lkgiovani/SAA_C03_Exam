# AWS SAA-C03 Quiz

Um questionário interativo de múltipla escolha para praticar para o exame **AWS Certified Solutions Architect – Associate (SAA-C03)**.

Feito para a comunidade, de forma simples e acessível — sem frameworks, sem dependências externas, só HTML, CSS e JavaScript puro.

---

## O que é

Este projeto transforma um banco de **710 questões** do exame SAA-C03 em uma aplicação web interativa com tema dark. As questões cobrem os principais domínios do exame:

- Design de arquiteturas resilientes
- Design de arquiteturas de alta performance
- Design de arquiteturas seguras
- Design de arquiteturas com custo otimizado

---

## Como usar

1. Clone ou baixe o repositório
2. Abra o arquivo `quiz.html` diretamente no navegador

> Os arquivos `quiz.html`, `quiz.css`, `quiz.js` e `questions.json` precisam estar na mesma pasta.

---

## Funcionalidades

- 710 questões extraídas do exame oficial
- Suporte a questões de resposta única e múltipla escolha
- Para questões com múltiplas respostas, o quiz limita a seleção ao número exato pedido
- Feedback imediato após confirmar a resposta
- Barra de progresso
- Placar ao final com percentual de acerto
- Modo aleatório e limite de questões configurável
